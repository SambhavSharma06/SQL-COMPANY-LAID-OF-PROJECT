# World Layoffs SQL Analysis

This project analyzes the World Layoffs dataset using MySQL. It covers the complete data analysis workflow, including data cleaning, transformation, and exploratory data analysis (EDA).

## Skills Demonstrated
- SQL
- Data Cleaning
- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- GROUP BY & ORDER BY
- Joins
- Business Insights

## Dataset
The dataset contains information on company layoffs, including company names, industries, countries, funding stages, dates, and the number of employees laid off.

## Objectives
- Clean and prepare raw data
- Analyze layoff trends over time
- Identify companies with the highest layoffs
- Compare layoffs by industry and country
- Generate meaningful business insights using SQL
